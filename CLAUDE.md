# supragnosis 작업 규칙

## 문서 작성 규칙: 부호는 ASCII만

문서, 주석, 커밋 메시지에서 구두점/기호는 ASCII만 사용한다.
(한글 등 본문 텍스트는 당연히 허용. "부호"만 해당.)

| 지양 (non-ASCII) | 사용 (ASCII) |
|------------------|--------------|
| em/en dash `—` `–` | ` - ` 또는 쉼표로 문장 재구성 |
| 가운뎃점 `·` | `,` 또는 `/` |
| curly quotes `“ ” ‘ ’` | `" "` `' '` |
| 말줄임 `…` | `...` |
| 화살표 `→` `←` `↔` | `->` `<-` `<->` |
| 곱셈/수학 `×` `≥` `≤` | `x` `>=` `<=` |
| 불릿 특수문자 `•` `▸` | `-` (마크다운 리스트) |
| 체크/이모지 `✅` `❌` | `[o]` `[x]` 또는 텍스트 |

- 기존 문서를 편집할 때 주변에 non-ASCII 부호가 보이면 함께 정리한다.
- 예외: 외부 인용문 원문, mermaid/코드가 요구하는 문법은 그대로 둔다.
