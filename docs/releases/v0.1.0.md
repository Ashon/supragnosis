The first release of an embedded, file-based Rust server that integrates knowledge from multiple
hosts/workspaces into an ontology (a concept/relation graph) **while preserving provenance** and
lets you query and explore it over **MCP**.

> `supragnosis` = *supra* (above/beyond) + *gnosis* (knowing) - knowledge above knowledge (meta-knowledge).

## Highlights

### MCP tools (LLM agent surface)
- `observe` - load a piece of knowledge as an immutable observation (the source of truth) and link its entities/relations into the ontology
- `search_knowledge` - semantic + keyword hybrid search (the response `mode` reports whether it degraded)
- `get_entity` - look up entity + relations + provenance (absence is not falsehood but the unknown - open-world assumption)
- `traverse` - n-hop traversal of the relation graph
- `workspace_map` - overview co-occurrence context (hyperedges) by name; orientation before searching
- Resources: ontology graph / hypergraph / observation back-reference (original text + provenance + lineage)

### Knowledge model
- **event sourcing** - the immutable observation log is the source of truth; the graph is a deterministic projection
- **provenance as a first-class citizen** - delegation chain + trust grade; content address (blake3) for path-independent dedup
- **hypergraph (second-order structure)** - revives the set of entities that a single observation jointly asserted,
  recovering the "what was said together" (context) that binary relations discard. A deterministic input for schema induction/resolution/consolidation
- **Cozo/RocksDB persistence** - unifies relational + graph + vector (HNSW), hexagonal port/adapter

### Execution modes
- **stdio MCP** - the agent launches it as a child process
- **standalone HTTP daemon** - a single always-on localhost daemon holds the db, shared by multiple agents
- **live ontology viewer** - a localhost canvas: hyperedge hull overlay, visibility toggle,
  edge colors by relation kind, search/traversal footprints

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/Ashon/supragnosis/main/scripts/install.sh | sh
```

Or download the platform `tar.gz` directly from the assets below, extract it, and put `supragnosis` on your PATH.

- Supported platforms: macOS (arm64 / x86_64), Linux (x86_64). Each asset includes a `.sha256` checksum.
- Registration example: `claude mcp add supragnosis -- ~/.local/bin/supragnosis`

## Notes / limitations

- The prebuilt binary is **keyword + hashing search**. For local ONNX **semantic search**, build from source with
  `cargo build --release --features fastembed` (excluded here because of the heavy ONNX runtime dependency).
- The HTTP daemon / viewer is **loopback-only (no auth)** - a local trust surface. Remote exposure / auth / TLS come later.
- **0.x early release** - the storage format and surface may change in the future.
